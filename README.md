<h3 id="objetivo">🎯 Goals</h3>

- Getting better at programming in the only way that works... by practicing
- In this project I used a fetch API to create an app that can find the lyric searching for the artist and the name of the song. I consumed the API by creating a function that receive two arguments: artist name and the name of the song. Then, the arguments pass to the url to be fetched. With that, the function return a promise and after a treatment I get an object with the lyric. The rest is quiet simple just some innerHTML and some adjustments.
- I used Bootstrap 5 to syle the app and just a fill adjustments with pure CSS.
- I pretend to add some more features in the future like a favbar to add your favorite songs.
